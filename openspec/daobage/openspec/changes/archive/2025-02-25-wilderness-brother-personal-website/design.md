## Context

荒野刀疤哥需要一个新的数字名片来展示其独特的人格魅力。参考史蒂夫·乔布斯官网的设计理念——极简、优雅、聚焦内容——我们将打造一个具有荒野气质的个人网站。

设计灵感来源：
- Steve Jobs 官网：大字体、留白、黑白对比、照片主导
- 荒野美学：大地色系、自然纹理、粗犷质感

## Goals / Non-Goals

**Goals:**
- 创建一个令人印象深刻的单页滚动网站
- 融合荒野的粗犷与极简的优雅
- 确保在各种设备上都有出色的阅读体验
- 快速加载，无多余依赖

**Non-Goals:**
- 不涉及后端或数据库
- 不需要 CMS 内容管理系统
- 不实现复杂的交互动画（保持简洁）
- 不支持多语言版本（仅中文）

## Decisions

### 技术栈选择：纯 HTML + CSS + 原生 JS

**选择原因**：
- 静态网站无需复杂框架
- 加载速度快，SEO 友好
- 易于部署到任何静态托管服务
- 与 Steve Jobs 官网的技术理念一致

**替代方案**：使用 React/Vue 框架 - 过于复杂，不符合极简理念

### 视觉风格：荒野极简主义

**设计元素**：
- 配色：深棕 (#3D2817)、赭石 (#8B4513)、米色 (#F5F0E6)、墨绿 (#2D3B2D)
- 字体：中文使用思源宋体（优雅）+ 中文无衬线（现代）
- 纹理：细微的纸张/岩石纹理背景
- 留白：大量留白，让内容呼吸

**选择原因**：
- 大地色系传递荒野、自然的质感
- 留白和字体选择致敬 Jobs 的极简美学
- 纹理增加层次感但不喧宾夺主

### 布局结构：单页滚动

**结构**：
1. Hero - 全屏视觉冲击
2. About - 人物故事
3. Philosophy - 三列理念卡片
4. Gallery - 图片网格
5. Contact - 极简联系区域

**选择原因**：
- 线性叙事，引导访客逐步了解
- 移动端体验一致
- 实现简单，维护成本低

### 图片策略：全屏背景 + 内容图片

**Hero 区域**：
- 全屏背景图（荒野风景或人物剪影）
- 渐变遮罩保证文字可读性

**Gallery 区域**：
- 网格布局展示 6-9 张精选图片
- 懒加载优化性能

**选择原因**：
- 视觉冲击力，符合 Wilderness 主题
- 类似 Jobs 官网的大图风格

## Risks / Trade-offs

| Risk | Mitigation |
|------|------------|
| 图片加载慢影响体验 | 使用图片压缩、WebP 格式、懒加载 |
| 移动端文字可读性 | 字体大小响应式调整，确保最小 16px |
| 大地色背景可能显得沉闷 | 使用高对比度白色文字，增加呼吸感 |
| 浏览器兼容性 | 使用现代 CSS 但保持渐进增强 |
